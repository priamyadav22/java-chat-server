# Java Chat Server

## Overview
This project is a simple HTTP-based chat server built in Java. It processes incoming requests and allows users to send messages that are displayed in a chat-like format.

## Features
- Handles HTTP requests using custom routing logic
- Supports adding messages via:
  - `/add-message?s=message&user=username`
- Parses query parameters to extract user input
- Dynamically generates chat output

## Technologies
- Java
- Networking fundamentals
- URI and query parsing

## What I Learned
- How servers handle requests and routes
- Parsing and processing user input from URLs
- Structuring backend logic for dynamic responses
